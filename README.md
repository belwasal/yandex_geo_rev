# Анализ пользовательской активности и тональности отзывов (Yandex Geo Reviews)

В данном репозитории представлена техническая реализация исследования пользовательского опыта на базе полумиллиона отзывов геосервиса. Цель - измерить лояльность аудитории, выявить поведенческие триггеры и предсказать тональность обратной связи.

### Технический стек и реализованные задачи:

* **NLP и Feature Engineering:** 

Анализ семантики с использованием TF-IDF для фильтрации информационного мусора. Программный парсинг морфологической структуры текста для доказательства корреляции между длиной отзыва, частотой использования спецсимволов и итоговой оценкой.
* **Машинное обучение (Regression & Classification):** 

Построение и сравнение пайплайнов для прогнозирования длины отзыва и его тональности. В коде реализовано тестирование широкого стека моделей: от базовых линейных алгоритмов до ансамблей (CatBoost, Decision Tree), метода опорных векторов (SVR с применением PCA) и полносвязных нейросетей (Multi-Layer Perceptron).
* **Метрики качества:** 
Оценка работы алгоритмов проводилась с использованием стандартных регрессионных метрик (MAE, RMSE, R2) и метрик классификации с учетом дисбаланса классов целевой переменной.
